Why Debian?

I have happily used Gentoo, Centos, Mint, and other distros going back to boot floppies. I've also used every iteration of Windows and MacOS, old school picks like SunOS, Solaris, HP-UX, AIX, IRIX, and even VMS. I don't hold strong opinions on operating systems (but editors are a different story - I didn't grow that extra meta-finger for nothing!)

I chose Debian because of numbers — 80% of the examples that I ran across tended to focus on Debian, and it is the base for over a hundred other downstream distros, like Ubuntu and Mint. I started with Debian 9 (Stretch) but moved to Debian 10 (Buster) shortly after it was released.

The Debian website has a catalog of all available packages, and a backports catalog with newer versions of certain packages.

I advise using Ansible for any additions, but while prototyping, occasional calls to the apt package manager come in handy:

apt list --installed       # all installed packages
apt list <pkg>
apt list <pkg> -a    # list all versions in the repo

# to completely uninstall and reinstall a package.
apt-get --purge remove <pkg>
apt-get install <pkg>
