Debian

Why Debian? When starting this work, I didn't really have strong opinions on a distro. In recent years, I have happily used Gentoo, Centos and Mint on a daily basis. The main reason is just numbers — 80% of the examples that I ran across tended to be for Debian, and it is the base for over a hundred other downstream distros, like Ubuntu and Mint. I started with Debian 9 (Stretch) but moved to Debian 10 (Buster) shortly after it was released.

The Debian website has a catalog of all available packages, and a backports catalog with newer versions of certain packages.

I advise using Ansible for any additions, but while prototyping, occasional calls to the apt package manager come in handy: 

apt list --installed       # all installed packages
apt list <pkg>
apt list <pkg> -a    # list all versions in the repo

# to completely uninstall and reinstall a package.
apt-get --purge remove <pkg>
apt-get install <pkg>
